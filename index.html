  • Git clone - клонировать репозиторий на локальную машину
  • Git status - получить инфу по статусу локальной копии репозитория
  • Git add - добавить новый файл под контроль/трекинг гита
  • Git commit -am "added new file index.html" - зафиксировать изменения и дать им комментарий
  • Git push - отправка закоммиченных изменений в репозиторий на сервер
  • Git pull - забрать свежие обновления
